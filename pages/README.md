# CT_B1 FLask app


## start virtualEnv 

```bash
cd <directory of the project>
```
**osx / linux**

```bash 
. ./venv/bin/activate
```

**windows**

```bash
\venv\Scripts\activate
```


## run flask

```bash 
flask --debug run
```


### REFERENCES 

*flask install*
